This entity represents a customer of the store, typically an individual person. A Customer can be
a guest, in which case it has no associated <DocsLink href="/reference/typescript-api/entities/user#user">User</DocsLink>. Customers with registered account will
have an associated User entity.

```ts title="Signature"

    @Column({ type: Date, nullable: true })

    title: string;
```
