A Role represents a collection of permissions which determine the authorization
level of a [User](/reference/typescript-api/entities/user#user) on a given set of [Channel](/reference/typescript-api/entities/channel#channel)s.

```ts title="Signature"
class Role extends VendureEntity implements ChannelAware {

    @Column() code: string;
    @Column() description: string;
```
