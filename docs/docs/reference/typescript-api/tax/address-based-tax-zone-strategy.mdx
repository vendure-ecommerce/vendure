Address based [TaxZoneStrategy](/reference/typescript-api/tax/tax-zone-strategy#taxzonestrategy) which tries to find the applicable [Zone](/reference/typescript-api/entities/zone#zone) based on the
country of the shipping address of the Order.
This is useful for shops that do cross-border B2C orders and use the One-Stop-Shop (OSS) VAT scheme.

Returns the default [Channel](/reference/typescript-api/entities/channel#channel)'s default tax zone if no applicable zone is found.

:::info

This is configured via `taxOptions.taxZoneStrategy = new AddressBasedTaxZoneStrategy()` in
your VendureConfig.

:::

*Example*

```ts
import { VendureConfig, AddressBasedTaxZoneStrategy } from '@vendure/core';

export const config: VendureConfig = {
  // other options...
  taxOptions: {
    taxZoneStrategy: new AddressBasedTaxZoneStrategy(), // [!code highlight]
  },
};
```
