### logToConsole

This can be useful for local development or debugging.

*Example*

```ts
import { LogLevel } from '@vendure/core';
import { TelemetryPlugin } from '@vendure/telemetry-plugin';

// ...

TelemetryPlugin.init({
  loggerOptions: {

});
```
